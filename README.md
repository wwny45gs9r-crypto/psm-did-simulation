# PSM-DID Monte Carlo Simulation

This repository contains a Monte Carlo simulation assignment for comparing PSM, DID, DID with covariates, and PSM-DID under different data-generating processes.

## Files

- `psm_did_simulation.ipynb`: Python code for simulation and visualization.
- `report.md`: Final report.
- `figures/`: Figures used in the report.
- `outputs/`: Simulation result tables.

## Methods

The project compares four estimators:

- PSM
- DID
- DID with covariates
- PSM-DID

The true treatment effect is set to tau = 2.
